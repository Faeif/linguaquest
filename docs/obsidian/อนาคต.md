# LinguaQuest — อนาคต (Phase 2+)

> ทุกอย่างในไฟล์นี้ **ยังไม่ทำ** — รอ validation phase ผ่านก่อน  
> เกณฑ์เข้า Phase 2: Day-7 retention ≥ 30% และ beta users ≥ 20 คน

## Phase 2 — หลัง Validation (พ.ค.–มิ.ย. 2026)

### 2.1 Real Talk Mode (เต็มรูปแบบ)
```
ตอนนี้: Learner mode อย่างเดียว
Phase 2: Real Talk — native speed, ไม่มี hint, ไม่มี hint card

เพิ่มเติมจาก MVP:
→ Difficulty scaling ตาม session history
→ Persona พูดแบบ overlap (interrupt) เหมือนคนจริง
→ Filler words (那个 就是 嗯) มากขึ้น
→ Score threshold สำหรับ unlock Real Talk
```

### 2.2 Grammar Card + Sentence Card
```
ตอนนี้: Flashcard มีแค่ Vocab type
Phase 2:
→ Grammar Card (จาก Grammar Pill → บันทึก flashcard)
→ Sentence Card (จาก Session End → บันทึก full sentences)
→ Review mode: Fill blank, Error correction, Role play
```

### 2.3 Session History + Replay
```
→ /progress/history — ดูทุก session ที่เคยทำ
→ [เล่นซ้ำ] — topic เดิม แต่ AI generate scenario ใหม่
→ ดู speaking score trend รายสัปดาห์
→ Vocab ที่เจอในแต่ละ session (Recently Seen UI)
```

### 2.4 Grammar History + Search
```
→ /learn/grammar → tab [ที่เรียนแล้ว]
→ ค้นหา grammar ที่เคยเจอได้
→ เรียงตาม: ล่าสุด / HSK Level / ผิดบ่อยสุด
→ [+ Flashcard] จาก grammar history
```

### 2.5 User-Designed Study Plan
```
→ [+ เพิ่มเป้าหมาย] — สร้าง topic เองได้
→ กำหนดวันที่เอง
→ AI generate lessons ตาม custom topic
→ Remix plan: ปรับ order ของ day/topic
```

## Phase 3 — ขยาย (ก.ค.–ส.ค. 2026)

### 3.1 Tutor Chat (General + HSK)
```
แชทแบ่งเป็น 2 โหมด:

1. General Chat — คุยเรื่องอะไรก็ได้
   → ไม่มี topic structure
   → AI ตอบแบบ conversational ตลอด
   → เหมาะสำหรับ intermediate+

2. HSK Tutor Chat — เตรียมสอบ
   → เน้น grammar + vocab ตาม HSK level
   → ทำ mock exam questions
   → AI อธิบายคำตอบเป็นภาษาไทย
   → Track HSK readiness score
```

### 3.2 Tutor Community
```
→ Forum/community สำหรับผู้เรียน
→ โพสต์คำถามภาษาจีน → Somnuek หรือ native คนอื่น answer
→ Upvote system สำหรับ best answer
→ Weekly challenge: แปลประโยคนี้ดูสิ
```

### 3.3 Knowledge Book
```
→ Curated articles เรื่อง Chinese culture/language
→ เชื่อมกับ vocabulary ในบทความ (tap → popup)
→ คัดโดย Somnuek + native speakers
→ Beginner / Intermediate / Advanced filter
```

### 3.4 Shadowing Mode
```
→ ฟัง AI พูดประโยค → พูดตาม real-time
→ Compare waveform: AI vs user
→ Pitch contour visualization (สูง/ต่ำ)
→ เหมาะสำหรับ tone improvement เฉพาะทาง
```

### 3.5 Listening Comprehension
```
→ ฟัง dialog (2 คน) → ตอบคำถาม
→ ทำได้ offline (pre-generated audio)
→ เนื้อหาเชื่อมกับ study plan topic
→ Difficulty: slow → normal → fast speed
```

## Phase 4 — Monetization (ก.ย. 2026+)

### Freemium Model (เบื้องต้น)
```
Free:
→ 10 AI messages/วัน
→ Flashcard (vocab) ไม่จำกัด
→ Study Plan (7 วัน)
→ 3 topics

Pro (฿149–199/เดือน):
→ Unlimited AI chat
→ Speaking History + Progress graph
→ HSK Readiness Score
→ Real Talk mode ไม่จำกัด
→ All topics (30+)
→ Grammar + Sentence Cards
→ Tutor Chat (HSK mode)

Family/Group: TBD หลัง individual plan stable
```

### Features ที่ convert ได้ดีสุด (ตามสมมติฐาน)
```
1. Unlimited AI Companion (free limit 10 msg รู้สึกน้อย)
2. Speaking Progress Graph (เห็น improvement = อยากต่อ)
3. HSK Readiness Score (คนสอบพร้อมจ่าย เพราะ goal ชัด)

→ ทดสอบก่อนตัดสินใจ paywall:
   ถามตรงๆ ใน willingness-to-pay survey
   "ถ้าต้องจ่าย feature ไหนที่คุณยอมจ่ายก่อน?"
```

## ไม่ทำ (จงใจ)
```
❌ Social features แบบ Duolingo (league, friend competition)
   เหตุผล: toxic gamification → burnout → churn
   แทนด้วย: leaderboard friends-only + supportive messages

❌ Grammar lecture ก่อนฝึกพูด
   เหตุผล: เบื่อ ไม่ effective
   แทนด้วย: เรียน grammar ผ่าน context (recast + grammar pill)

❌ Subscription แบบ auto-lock progress (pay or lose streak)
   เหตุผล: dark pattern ทำลาย trust
   แทนด้วย: ใช้ต่อได้ แค่ feature บางอย่าง unlock ด้วย pro

❌ Translate mode (แปลไทย↔จีน)
   เหตุผล: นอก scope — มี Google Translate แล้ว
```
